Use Geth to Create a Prive Network

Introduction

繼上篇成功在testrpc上創建private network、轉帳、部署智能合約及智能合約外，這篇要教學的是如何透過geth 本身，創建private network，並在上面進行轉帳等功能。

Geth 與 testrpc架設private network差別在於：

private network 創建帳戶 帳戶後是否有錢 解鎖帳戶 挖礦
Geth 需手動添加 無 需要解鎖帳戶才可轉帳 需手動挖礦
testrpc 自動添加十組給你 預設有一筆錢 不需 自動挖礦

本章目標：

  • Create a private network(geth)

  • Use geth to link the private network

  • Use Web3.js API to query account’s balance and transfer ether
